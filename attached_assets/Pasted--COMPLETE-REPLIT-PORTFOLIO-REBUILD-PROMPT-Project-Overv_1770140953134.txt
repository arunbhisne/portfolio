## Technical Requirements

### Framework & Stack
- **Framework:** React with Next.js (or modern React with Vite)
- **Styling:** Tailwind CSS + custom CSS variables for design system
- **Animations:** Framer Motion for smooth interactions
- **Icons:** Lucide React or Heroicons
- **Performance:** Code splitting, lazy loading, optimized images (WebP/AVIF)
- **Responsiveness:** Mobile-first approach (breakpoints: 375px, 768px, 1024px, 1440px)
- **Accessibility:** WCAG AA compliant (contrast ratios, alt text, keyboard navigation, ARIA labels)

### Performance Targets
- Lighthouse score: 90+ on mobile and desktop
- First Contentful Paint: < 1.5s
- Largest Contentful Paint: < 2.5s
- Cumulative Layout Shift: < 0.1
- Time to Interactive: < 3.5s

## Design System

### Color Palette
**Light Mode:**
- Background: `#FAFAF9` (warm off-white)
- Surface: `#FFFFFF` (pure white cards)
- Text Primary: `#1A1A1A` (near black)
- Text Secondary: `#6B7280` (gray-500)
- Accent Primary: `#14B8A6` (teal-500 - represents AI/tech)
- Accent Hover: `#0D9488` (teal-600)
- Border: `#E5E7EB` (gray-200)

**Dark Mode:**
- Background: `#0A0A0A` (true black)
- Surface: `#1A1A1A` (dark gray cards)
- Text Primary: `#F5F5F5` (off-white)
- Text Secondary: `#9CA3AF` (gray-400)
- Accent Primary: `#2DD4BF` (teal-400)
- Accent Hover: `#14B8A6` (teal-500)
- Border: `#374151` (gray-700)

### Typography
- **Font Family:** Inter (primary), Space Grotesk (headings alternative), JetBrains Mono (code)
- **Scale:**
  - H1: `clamp(2.5rem, 5vw, 3.5rem)` - Bold (700)
  - H2: `clamp(2rem, 4vw, 2.75rem)` - Semibold (600)
  - H3: `clamp(1.5rem, 3vw, 2rem)` - Semibold (600)
  - H4: `clamp(1.25rem, 2.5vw, 1.5rem)` - Medium (500)
  - Body: `1rem (16px)` - Regular (400)
  - Small: `0.875rem (14px)` - Regular (400)
- **Line Height:** 1.6 for body text, 1.2 for headings
- **Letter Spacing:** -0.02em for headings

### Spacing & Layout
- **Grid System:** 8px base unit
- **Container Max Width:** 1200px
- **Section Padding:** 120px vertical (desktop), 80px (tablet), 60px (mobile)
- **Card Padding:** 32px (desktop), 24px (mobile)
- **Gaps:** 24px between cards, 16px between elements

### Shadows & Effects
- **Card Shadow:** `0 1px 3px rgba(0,0,0,0.05), 0 4px 12px rgba(0,0,0,0.03)`
- **Card Hover:** `0 4px 6px rgba(0,0,0,0.07), 0 12px 24px rgba(0,0,0,0.05)`
- **Border Radius:** 12px (cards), 8px (buttons), 6px (inputs)
